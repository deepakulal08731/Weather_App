# 🌤️ Weather App

A simple weather app that shows real-time temperature, humidity, and wind speed for any city using the OpenWeatherMap API.


## 🚀 Features

- Search for any city
- Shows:
  - Temperature
  - Humidity
  - Wind speed
- Changes weather icon dynamically
- Displays error if city not found

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript
- OpenWeatherMap API
